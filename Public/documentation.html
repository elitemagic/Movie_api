myFlix is a server-side web application, providing users with access to information about differnt movies, actors, directors and their specific genre. ENabling users to sign up, update their personal information and create list of their favourite movies.


The following table displays a list of features myFlix can preform

Request URL Response
Return a list of ALL movies to the user /movies JSON object holding data about all the movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/Tommy%20Boy JSON object holding data about the movie Tommy Boy
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genre/Comedy JSON object holding data about the genre comedy
Return data about a director (bio, birth year, death year) by name /movies/Directors/James%20Cameron JSON object holding data about director James Cameron
Allow new users to register /users JSON object holding data about a new user that was added
Allow users to update their user info (username) /users/1 JSON object holding data of updated user name
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/2/Tommy%20Boy JSON object holding data of users updated favorite movie - Tommy Boy
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/2/Tommy%20Boy JSON object
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users/2 Text describing user being removed
